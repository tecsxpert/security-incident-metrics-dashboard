AI Endpoints
POST /describe
Analyzes a security incident and returns a structured breakdown — summary, attack vector, impact, indicators, and severity justification.
Returns: summary, attack_vector, impact, indicators, severity_justification
POST /recommend
Generates exactly 3 prioritized remediation recommendations — containment, eradication, and prevention — for any security incident.
Returns: 3 recommendations [ action_type, description, priority ]
POST /generate-report
Produces a complete professional incident report — executive summary, timeline, technical analysis, impact assessment, and remediation plan.
Returns: full report with timeline, analysis, impact, remediation
GET /health
Service health check with model info, average response time, uptime, and Redis cache status.
Returns: status, model, avg_response_time, uptime, cache_enabled
